Perkenalan Singkat Tentang Diri Saya

Halo Semuanya, nama saya Wahyu Nahda Putra. Saya adalah seorang mahasiswa, editor, dan penggemar perangkat lunak gratis.

Saya tinggal di Kota Kendari, Sulawesi Tenggara, Indonesia. Saya lahir di Kota Kendari, 20 April 2003. Umur saya 19 tahun. Dan saya seorang Mahasiswa Teknik Informatika di Universitas Halu Oleo, Kendari.

Bahasa Pemrograman yang saya pelajari, yakni PHP dan JavaScript. Bahasa marka HTML, bahasa desain CSS, dan basis data/DBMS MYSQL. Framework Codeigniter, Bootstrap 4, dan Tailwind CSS.

Saya mulai mempelajari dan menyukai coding ketika masuk di Sekolah Menengah Kejuruan, Jurusan Rekayasa Perangkat Lunak atau yang familiar disebut dengan RPL.
